¿Cuándo se lanzará HTML 6? ¿Qué mejoras respecto a HTML5?

HTML es el lenguaje básico de la web, utilizado para estructurar y mostrar el contenido de las páginas web. Desde su creación en 1991, HTML ha experimentado varias evoluciones, con la incorporación de nuevas etiquetas, nuevos atributos y nuevas funciones.

La última versión principal del lenguaje, HTML5, se lanzó en 2014 y trajo muchas mejoras.

¿Cuándo podemos esperar que llegue HTML 6? ¿Qué habrá de nuevo en esta versión? Esto es lo que intentaremos ver en este artículo.

¿Qué es HTML6?

Este es un proyecto en curso que tiene como objetivo ampliar las capacidades del HTML5 actual y satisfacer las necesidades emergentes de la web.

HTML6 aún no es un estándar oficial, sino un conjunto de propuestas y experimentos discutidos por la comunidad web.

Entre estas iniciativas podemos citar:

  • Web Components: permite crear elementos HTML personalizados y reutilizables.
  • Proyecto Houdini: ofrece más control sobre cómo se representan los elementos en el navegador.
  • Extensible Web Manifesto: busca que los estándares web sean extensibles por los propios desarrolladores.

¿Cuándo llegará HTML6?

HTML ahora evoluciona bajo un modelo "vivo", con actualizaciones incrementales en lugar de versiones completas.

Muchas características pensadas para HTML6 ya se están incorporando poco a poco en los navegadores modernos.

Otras ideas pueden ser abandonadas o reemplazadas por soluciones más efectivas.

Herramientas como HTML6.js y HTML6 Boilerplate permiten experimentar hoy con propuestas del futuro.

¿Cuál es la diferencia entre HTML6 y HTML5?

HTML6 buscará una sintaxis más moderna, modular y adaptable, mejor integración con tecnologías web actuales y mayor soporte multiplataforma.

HTML5 ya fue un gran avance, incorporando soporte para audio, video, SVG, formularios mejorados, almacenamiento local y más.
