¿Cuándo se lanzará HTML 6? ¿Qué mejoras respecto a HTML5?

HTML es el lenguaje básico de la web, utilizado para estructurar y mostrar el contenido de las páginas web. Desde su creación en 1991, HTML ha experimentado varias evoluciones, con la incorporación de nuevas etiquetas, nuevos atributos y nuevas funciones.
La última versión principal del lenguaje, HTML5, se lanzó en 2014 y trajo muchas mejoras.

¿Pero qué pasa con el próximo? ¿Cuándo podemos esperar que llegue HTML 6 ? ¿Qué habrá de nuevo en esta versión? Esto es lo que intentaremos ver en este artículo.

¿Qué es HTML6?

Este es un proyecto en curso que tiene como objetivo ampliar las capacidades del HTML5 actual y satisfacer las necesidades emergentes de la web.

HTML6 aún no es un estándar oficial , sino más bien un conjunto de propuestas y experimentos que son probados y discutidos por la comunidad web . Tampoco existe una única especificación para HTML6, sino varias iniciativas que pretenden enriquecer el lenguaje en diferentes aspectos .

Entre estas iniciativas podemos citar:

  • El proyecto Web Components , que proporciona una manera de crear elementos HTML personalizados y reutilizables con su propia estructura, estilo y comportamiento.
  • El Proyecto Houdini , que tiene como objetivo brindar a los desarrolladores más control sobre la representación de elementos HTML y CSS, al exponer las API internas del motor de representación del navegador.
  • El proyecto Extensible Web Manifesto , que defiende la idea de que los estándares web deben diseñarse de manera que permitan a los desarrolladores ampliar la funcionalidad de la web con sus propias innovaciones.

Estos proyectos no forman parte de HTML6 estrictamente hablando, pero suelen estar asociados a esta idea de evolucionar el lenguaje hacia una mayor modularidad, flexibilidad y creatividad.

¿Cuál es la diferencia entre HTML6 y HTML5?

HTML6 es la próxima versión de HTML , sucediendo a HTML5. HTML6 tiene como objetivo simplificar y modernizar el lenguaje HTML, ofreciendo una sintaxis más concisa y expresiva, mejor integración con otras tecnologías web, mayor modularidad y extensibilidad, y mejor compatibilidad con diferentes navegadores y dispositivos.

HTML5 es la versión actual de HTML , lanzada en 2014. HTML5 ya trae muchas mejoras y características en comparación con versiones anteriores, como soporte para medios (audio, video, etc.), dibujo vectorial (SVG), formularios interactivos, almacenamiento local, geolocalización, etcétera.
